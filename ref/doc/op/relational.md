# Relational Operators

| Operator | Type  | Description              |
| :------: | ----  | -----------              |
| `==`     | Infix | Equality                 |
| `!=`     | Infix | Inequality               |
| `<`      | Infix | Less-than                |
| `<=`     | Infix | Less-than or equal-to    |
| `>`      | Infix | Greater-than             |
| `>=`     | Infix | Greater-than or equal-to |
