# Bitwise Operators

| Operator | Type   | Description         |
| :------: | ----   | -----------         |
| `&`      | Infix  | Bitwise `AND`       |
| `|`      | Infix  | Bitwise `OR`        |
| `^`      | Infix  | Bitwise `XOR`       |
| `<<`     | Infix  | Bitwise left shift  |
| `>>`     | Infix  | Bitwise right shift |
| `~`      | Prefix | Bitwise `NOT`       |
